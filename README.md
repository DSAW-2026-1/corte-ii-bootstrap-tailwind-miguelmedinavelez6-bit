## Overview
This project contains a responsive stats preview card component built in two versions:
- Bootstrap
- Tailwind CSS

The layout adapts to mobile, tablet, and desktop screens.

## Pages
- `index.html`
- `bootstrap.html`
- `tailwind.html`

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- Tailwind CSS

## Responsive Design
The project was tested for:
- Mobile Portrait (320px x 480px)
- Tablet Portrait (768px x 1024px)
- Desktop (+1024px)
